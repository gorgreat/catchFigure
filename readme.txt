Игра "Поймай фигуру" на чистом js. ПО мотивам уроков webformyself.com

Задача - отработка пройденного материала.

Суть игры
По игровому полю появляется фигура - задача на нее нажать. 


Алгоритм
1. Игровое поле
2. Возможность выбора фигруры - круг, квадрат, прямоугольник
3. Ввод времени игры в секундах
4. Рандомная смена положения фигуры и ее цвета
5. Подсчет удачных кликов
6. Вывод результатат
